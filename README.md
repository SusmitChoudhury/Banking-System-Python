# Banking-System-Python
A simple and interactive command-line banking system built using Python Object-Oriented Programming (OOP).
This project allows users to credit money, debit money, check balance, and maintain a minimum balance requirement.
It is a beginner-friendly Python project that demonstrates classes, objects, functions, and conditional logic.

Features

  Create Bank Account with account number, name, and starting balance

  Credit (Deposit) money into the account

  Debit (Withdraw) money with minimum ₹500 required

  Check balance anytime

  Interactive loop system

  Prevents invalid or unsafe transactions

  Clean and easy-to-understand Python OOP code
